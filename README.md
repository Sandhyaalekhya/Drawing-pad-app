# Drawing Pad Web App 🎨

An interactive web-based drawing application that allows users to create digital sketches in real-time.

# Features
- **Interactive Canvas:** Smooth drawing experience using mouse or touch inputs.
- **Customizable Tools:** Easily adjust brush size and color.
- **Canvas Management:** Clear button to reset the drawing area.
- **Responsive UI:** Optimized for various screen sizes and browsers.

# Technologies Used
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Canvas API)
- **Development Tool:** Visual Studio Code
- **Version Control:** Git & GitHub
